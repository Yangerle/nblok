# nblok
**this is a game**
>点击链接在web端游玩：[https://yangerle.github.io/nblok/src](https://yangerle.github.io/nblok/src){:target="_blank"}

七巧板游戏是中国古典游戏之一。
七块具有数学规律的板与你的大脑相连接。
创造出变幻莫测的图形。
激发了人的想象力。
古人的智慧可见一斑。
## 我从七巧板里得到的智慧
**如果你善于思考**
万物的发展皆由简至繁  
无论是一艘航母，还是一个网站，一台电脑，都是由基础的东西与人类的智慧拼接而成。这启示我们在学习时：
>既要站在巨人的肩上
>也要掌握核心科技，虽然你所处的世界很复杂，但你要明白他的核心与本质
>这样你才有思考的空间
